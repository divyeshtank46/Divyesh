JavaScript Array Methods

Learn them once and remember them forever with real-life analogies!

The Ultimate Analogy: Your Bookshelf

Imagine you have a bookshelf (your array) with books (your array elements). Let's see how array methods help you manage your books!

üí° Key Things to Remember:

  • Original Array: Some methods CHANGE it, others DON'T
  • Return Value: Always check what the method RETURNS
  • Callback Function: Most methods take a function that works on each element
forEach()

What it does: Executes a function for each element

üìö Bookshelf Analogy: You take each book off the shelf, look at it, and put it back
const books = ['Harry Potter', 'Lord of the Rings', 'Game of Thrones'];
books.forEach(book => console.log(book));
// Output: Harry Potter, Lord of the Rings, Game of Thrones
Returns: undefined
map()

What it does: Creates a NEW array by transforming each element

üìö Bookshelf Analogy: You take each book and put a fancy cover on it, creating a new set of covered books
const books = ['Harry Potter', 'Lord of the Rings'];
const coveredBooks = books.map(book => `Covered: ${book}`);
// coveredBooks = ['Covered: Harry Potter', 'Covered: Lord of the Rings']
Returns: NEW array
filter()

What it does: Creates a NEW array with elements that pass a test

üìö Bookshelf Analogy: You go through your books and only keep the fantasy books, removing others
const books = ['Harry Potter', 'Math Textbook', 'Lord of the Rings'];
const fantasyBooks = books.filter(book => book.includes('Harry') || book.includes('Rings'));
// fantasyBooks = ['Harry Potter', 'Lord of the Rings']
Returns: NEW array
find()

What it does: Returns the FIRST element that passes a test

üìö Bookshelf Analogy: You look for the first book with "Harry" in the title
const books = ['Harry Potter', 'Lord of the Rings', 'Harry Potter 2'];
const found = books.find(book => book.includes('Harry'));
// found = 'Harry Potter'
Returns: First matching element or undefined
findIndex()

What it does: Returns the INDEX of the first element that passes a test

üìö Bookshelf Analogy: You find the position (shelf number) of the first Harry Potter book
const books = ['Lord of the Rings', 'Harry Potter', 'Game of Thrones'];
const index = books.findIndex(book => book.includes('Harry'));
// index = 1
Returns: Index number or -1
some()

What it does: Checks if AT LEAST ONE element passes a test

üìö Bookshelf Analogy: You check if there's ANY fantasy book on your shelf
const books = ['Math Textbook', 'Science Book', 'Harry Potter'];
const hasFantasy = books.some(book => book.includes('Harry'));
// hasFantasy = true
Returns: true or false
every()

What it does: Checks if ALL elements pass a test

üìö Bookshelf Analogy: You check if EVERY book on your shelf is a fantasy book
const books = ['Harry Potter', 'Lord of the Rings'];
const allFantasy = books.every(book =>
  book.includes('Harry') || book.includes('Rings')
);
// allFantasy = true
Returns: true or false
reduce()

What it does: Reduces the array to a SINGLE value

üìö Bookshelf Analogy: You count the total number of pages in all your books
const pages = [320, 450, 280];
const totalPages = pages.reduce((sum, pageCount) => sum + pageCount, 0);
// totalPages = 1050
Returns: Single value
sort()

What it does: Sorts the elements (CHANGES original array)

üìö Bookshelf Analogy: You rearrange your books alphabetically by title
const books = ['Z Book', 'A Book', 'M Book'];
books.sort();
// books = ['A Book', 'M Book', 'Z Book']
Returns: SAME array (modified)
slice()

What it does: Returns a COPY of part of the array

üìö Bookshelf Analogy: You take photos of books from position 2 to 4
const books = ['Book1', 'Book2', 'Book3', 'Book4', 'Book5'];
const slice = books.slice(1, 4);
// slice = ['Book2', 'Book3', 'Book4']
Returns: NEW array
splice()

What it does: Adds/removes elements (CHANGES original array)

üìö Bookshelf Analogy: You remove 2 books from position 1 and add 3 new books there
const books = ['Book1', 'Book2', 'Book3', 'Book4'];
books.splice(1, 2, 'NewBook1', 'NewBook2', 'NewBook3');
// books = ['Book1', 'NewBook1', 'NewBook2', 'NewBook3', 'Book4']
Returns: Array of removed elements
concat()

What it does: Combines arrays into a NEW array

üìö Bookshelf Analogy: You combine books from two different bookshelves into one big shelf
const fantasy = ['Harry Potter', 'Lord of the Rings'];
const scifi = ['Dune', 'Foundation'];
const allBooks = fantasy.concat(scifi);
// allBooks = ['Harry Potter', 'Lord of the Rings', 'Dune', 'Foundation']
Returns: NEW array
